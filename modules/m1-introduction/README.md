# M1 - Introduction to Microservices

## Learning Objectives

By the end of this module you will be able to:

- Understand microservice architectures
- Compare monolithic and distributed systems
- Identify the advantages of microservices

---

## Monolithic Architecture

Traditional monolithic applications contain all components inside a single deployable unit.

Main disadvantages:

- Difficult scalability
- Poor fault isolation
- Complex maintenance

---

## Microservice Architecture

Microservices split the application into independent services.

Advantages:

- Independent deployment
- Better scalability
- Fault isolation
- Technology independence
